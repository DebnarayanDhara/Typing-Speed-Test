# Typing Speed Test

A simple command-line Typing Speed Test built with Python. The application measures typing speed, accuracy, total words typed, and displays the last test result using randomly selected paragraphs from a text file.

## Features

* Random Paragraph Selection
* Typing Speed (WPM) Calculation
* Accuracy Calculation
* Character Count
* Word Count
* Last Result
* File Handling
* Error Handling
* Clean Command-Line Interface

## Technologies Used

* Python 3
* Random Module
* Time Module
* File Handling

## How to Run

Clone the repository:

```bash
git clone https://github.com/DebnarayanDhara/Typing-Speed-Test.git
```

Go to the project folder:

```bash
cd typing-speed-test
```

Run the program:

```bash
python main.py
```
